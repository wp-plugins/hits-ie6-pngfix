Adds IE6 Compatability for PNG transparency, courtesy of http://www.twinhelix.com/css/iepngfix/

== Description ==

Upon installation and activation, this plugin will add the necessary code to your pages to take advantage of the TwinHelix approach to getting the IE6 PNG Transparency issue solved.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload folder `hits-pngfix` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Test and enjoy!
